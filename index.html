        pass  # This is called when the object is garbage collected

The class of a class

So what is the class of a class, or in other words: What is the metaclass?

print(type(Foo))
>> type

The metaclass

As the metaclass, type is the default object that defines how a class is created in Python; Just as a class defines how an object is created.

type(name, bases, dict) can also be used to create new classes when used as a class on its own. The type(obj), showing the class of an object, behaviour is just available for backward compatibility.
